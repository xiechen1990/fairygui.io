FairyGUI 编辑器

当前版本3.4.3


Windows平台支持简体中文、繁体中文和英文

OSX平台支持简体中文、繁体中文和英文

FairyGUI编辑器是绿色软件，解压后可直接使用，无需安装。建议解压到非中文目录下。

旧版本下载通道：

  • Windows res.fairygui.com/FairyGUI-Editor_X_X_X.zip
  • OSX res.fairygui.com/FairyGUI-Editor_Mac_X_X_X.zip

将XXX换成版本号的三个数字就可以了。
